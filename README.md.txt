Burnout Prediction Model

 Description
This project uses machine learning to predict employee burnout based on factors such as working hours, mental fatigue score, and company type. The goal is to help identify burnout risk and improve workplace productivity.

Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow
- streamlit
- joblib

 Project Structure
- data/ → dataset used
- model/ → trained model file
- src/ → source code
- requirements.txt → dependencies

 How to Run the Project

1. Clone the repository:
   git clone https://github.com/khadijatidris02-code/burnout-prediction.git

2. Navigate into the folder:
   cd burnout-prediction

3. Install dependencies:
   pip install -r requirements.txt

4. Run the project:
   python main.py

 Sample Prediction

Input:
- Working Hours: 9  
- Mental Fatigue Score: 7.5  

Output:
- Burnout Level: High

Results & Insights
The model shows that higher working hours and increased mental fatigue significantly contribute to burnout levels. This aligns with real-world expectations in workplace productivity analysis.
